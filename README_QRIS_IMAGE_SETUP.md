# Setup Gambar QRIS di Checkout

## Tujuan
Menampilkan **gambar QRIS** yang disimpan sendiri agar customer dapat langsung scan untuk pembayaran.

## Cara pakai
1. Siapkan file gambar QRIS kamu (misal):
   - `diamond-store/images/qris.png`
2. Pastikan nama file sama seperti yang dipanggil di kode.
3. Checkout akan menampilkan QRIS ketika user memilih metode **QRIS**.

## Catatan
- Gambar QRIS sebaiknya ukuran lebar (contoh 400x400 atau 600x600) dan format PNG/JPG.
